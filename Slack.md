## Core Navigation (Move Fast Across Slack)

| **Action** | **Shortcut (Windows)** | **What it Does** |
|-----------|----------------------|----------------|
| **Open Quick Switcher** | Ctrl + K | Jump to any channel, DM, or file instantly |
| **Switch channels (next/prev)** | Ctrl + Shift + ] / [ | Move between channels |
| **Go to DMs** | Ctrl + Shift + K | Open Direct Messages |
| **Go to Activity** | Ctrl + Shift + A | Mentions & notifications |
| **Go to Threads** | Ctrl + Shift + T | All threads you're in |
| **Go to Saved items** | Ctrl + Shift + S | Your saved messages |
| **Back / Forward** | Alt + ← / → | Navigate history |

## Messaging & Composition (Where You Gain Speed)

| **Action** | **Shortcut** | **What it Does** |
|-----------|-------------|----------------|
| **New message** | Ctrl + N | Start a new DM |
| **Reply in thread** | Ctrl + Shift + Enter | Send to thread |
| **Send message** | Enter | Send message |
| **New line** | Shift + Enter | Line break |
| **Edit last message** | ↑ | Quickly fix last message |
| **Delete message** | Delete (on hover) | Remove message |
| **Mark all read** | Ctrl + Shift + A | Clear notifications |
