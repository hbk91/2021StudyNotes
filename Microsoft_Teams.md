# Microsoft Teams Keyboard Shortcuts (Windows)

## Search Shortcuts (Very Important)

| **Action** | **Shortcut** |
|------------|--------------|
| **Global Search (top search bar)** | `Ctrl + E` |
| **Search Messages (across Teams & Chats)** | `Ctrl + Shift + F` |
| **Search in Current Channel** | `Ctrl + F` |
| **Search Commands** | `Ctrl + /` |

### How these differ (this matters):
- **`Ctrl + E`**  
  - Global search bar  
  - Finds **people, chats, teams, files, commands**
- **`Ctrl + Shift + F`**  
  - Message-only search  
  - Best for **“Who said this?” / keyword hunting**
- **`Ctrl + F`**  
  - Searches **within the current chat or channel only**

## Power User Search Tricks (Keyboard-First)

Use these **inside the search bar (`Ctrl + E`)**:

| **Query** | **What it Does** |
|----------|------------------|
| `/goto Team Channel` | Jump directly to a channel |
| `/chat Name` | Open a chat instantly |
| `/call Name` | Start a call |
| `/files` | Search files only |
| `/saved` | Show saved messages |
| `/unread` | Jump to unread messages |
